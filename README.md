# socks5-python
# 背景
- 工作中A、B两个区域的网络不通，需本程序打通，因此写出来方便大家使用
- 网上异步的socks5写的不够“好”，因此写了一个
# 依赖
- python >= 3.9
- pip3 install requirements.txt
# 运行  
- python3 main.py

默认：
用户名：admin
密码：admin
端口：1080

- python3 main.py --host=0.0.0.0 --port=10800 --username=admin --password=admin

# 附录
